# student-marks-prediction
Student Marks Prediction is a Machine Learning project using Python and Linear Regression to predict student marks based on study hours. It uses Pandas, Scikit-learn, and Matplotlib for data processing, model training, prediction, and visualization.
# Student Marks Prediction ML Project

## Overview
This machine learning project predicts student marks based on study hours using Linear Regression.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Dataset
Dataset contains study hours and marks scored by students.

## Algorithm Used
Linear Regression
